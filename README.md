## 📖 Overview

**MediCare** is a frontend healthcare platform that allows users to:

* Book doctor appointments
* Track medications
* Receive specialist recommendations

Built as a static web app, it runs entirely in the browser using `localStorage` — no backend required.

---

## ✨ Features

* 👤 User authentication (localStorage-based)
* 📅 Appointment booking & management
* 💊 Medication tracking
* 🤖 AI-powered specialist recommendations
* 🌙 Dark / Light mode
* 📱 Fully responsive design
* 🔒 Protected dashboard routes
